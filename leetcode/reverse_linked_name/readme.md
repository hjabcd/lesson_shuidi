# 206 反转链表

两种解法 三种语言

反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

# 通过前驱结点及后继结点的概念，使用遍历
    当前结点 next 指向它的前驱结点
    之前的前驱结点变成当前结点
    当前结点变成原结点的后继结点
    如果为null 则完成遍历

# 递归
    - 把复杂的变成简单的重复
    - 退出条件
        null时退出
    简单公式 n 个结点反转  1 个结点
    当前结点cur next 指针指向的结点的next 指向当前结点的话， 就反转了
    结点next 属性消除
